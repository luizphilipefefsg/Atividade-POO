# Controle de Biblioteca - CEET Giuseppe Altoé

Este projeto é uma aplicação de console desenvolvida em C# para a disciplina de Linguagem de Programação Orientada a Objetos. O objetivo é digitalizar o controle de empréstimos da biblioteca escolar.

## 📋 Funcionalidades
- Cadastro de Livros (Título, Autor, Ano).
- Cadastro de Alunos (Nome, Matrícula, Turma).
- Controle de disponibilidade para empréstimo.
- Listagem completa de acervo utilizando `List`.

## 🛠️ Tecnologias Utilizadas
- Linguagem C#
- .NET SDK
- Conceitos de POO: Classes, Objetos, Atributos, Métodos e Construtores.

## 🚀 Como executar
1. Abra o projeto no Visual Studio.
2. Compile e execute o projeto (F5).
3. O console exibirá a lista de livros e o aluno responsável pelo empréstimo atual.
